# documentation

# video module

This module handles the play back and manipulation of video. Useful for VJ'ing video loops and adding video fx.

# the video foldre

You must add your .mp4 video's to the "videos" folder.

# video commands

```
v on
v off

v <index>

v seq <1/0>

v random <1/0>

v fx puzzle 2
```
